Introducing Nautobot

Nautobot is a Network Source of Truth and Network Automation Platform. It's an open-source project that provides a single place to store and manage all your network data.

üìä Nautobot's Data Model

Nautobot organizes network data into logical objects with relationships. Here's how Alex's CSV data maps to Nautobot models:

CSV Column Nautobot Model Description
hostname Device.name The device's hostname
device_type Device.device_type Reference to DeviceType model
location Device.location Reference to Location model
ip_address IPAddress Assigned to device interface
status Device.status Active, Planned, Staged, etc.
