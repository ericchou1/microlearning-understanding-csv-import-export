PHASE 4: JOB PROTOCOL OVERVIEW

> UNDERSTANDING NAUTOBOT JOBS

In the Nautobot system, Jobs are executable Python scripts that can manipulate the database. You'll be writing custom Jobs to import and export CSV data.

jobs/example_job.py 
from nautobot.apps.jobs import Job, register_jobs

class MyJob(Job):
    """A basic Job template."""
    
    class Meta:
        name = "My Custom Job"
        description = "Does something useful"
    
    def run(self):
        """Main execution method."""
        self.logger.info("Job executing...")
        return "Job completed!"

register_jobs(MyJob)
