THE NAUTOBOT JOB PROTOCOL

In Nautobot, Jobs are the key to everything. Executable Python scripts that can read data, write data, transform it. If you want to manipulate CSV streams, you need to master Jobs.

jobs/basic_template.py 
from nautobot.apps.jobs import Job, register_jobs

class MyJob(Job):
    """Every Job needs a purpose."""
    
    class Meta:
        name = "My Custom Job"
        description = "What this Job does"
    
    def run(self):
        """Main execution. This is where the magic happens."""
        self.logger.info("Job executing...")
        return "Success"

register_jobs(MyJob)  # Make it visible to Nautobot
